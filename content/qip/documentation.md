<p>Educators feel challenged by the workload completing individual documentation for all children as well as programming and planning for the classroom and outdoor learning environments. Educators feel some documentation is unnecessary and not significant to children.</p>
<p>Outcomes</p>
<ul>
<li>Documentation is meaningful and significant to children's individual learning journey</li>
<li>Workload for educators is reduced</li>
<li>Increased involvement/input from families</li>
<li>Clear links between documentation and children's goals</li>
<li>Clear planning cycle</li>
<li>Documentation will continue to support our philosophy, EYLF &amp; NQS</li>
<li>Quality and quantity of documentation is consistent across the service</li>
<li>Documentation is differentiated to suit different age groups</li>
</ul>
<p>Short Term Strategies</p>
<ul>
<li>Discussions with educators</li>
<li>Research alternatives to current documentation</li>
<li>Survey to families</li>
<li>Differentiate documentation for age groups</li>
<li>Identify possible changes</li>
<li>Trial changes and review</li>
</ul>
<p>Long Term Strategies</p>
<ul>
<li>Identify best quality outcomes and implement changes</li>
<li>Update Orientation document</li>
<li>Update documentation overviews in staff manual</li>
</ul>
